There are several padding related properties and values that we can take advantage of. The key thing to remember is that padding is applied INSIDE an element's border, whether that border is visible or not.

This outer div has padding added on all sides, with a border applied to make it easier to see how padding is applied.

When you provide one value to the padding property, that value gets applied to all sides of the element's Box Model.

padding: all-sides;

This div has padding added on all sides, but with different top/bottom and left/right values, as well as a border applied to make it easier to see how padding is applied.

When you provide two values to the padding property, the first value gets applied to the top and bottom of the Box Model, while the second value is applied to the left and right sides.

padding: top/bottom left/right;

This div has padding added on all sides, but with different top, left/right, and bottom values, as well as a border applied to make it easier to see how padding is applied.

When you provide three values to the padding property, the first value gets applied to the top of the Box Model, the second value is applied to the left and right sides, and the third value is applied to the bottom.

padding: top left/right bottom;
