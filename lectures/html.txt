HTML Syntax (Grammar)

HTML Tag

Three parts:
- opening triangle bracket
- tag name
- closing triangle bracket

example: <body>


HTML Element

Three parts:
- opening tag
- content/children 
- closing tag (has a slash in front of the tag name)

example 1: <p>This is some sample content</p>

example 2:
<p>
    <b>This text is bold</b>
    <i>This text is italicized</i>
</p>
