Setting up Pantheon and your first (live) WordPress site

  1. Create an account for Pantheon here.
  2. Set up your new Pantheon site, by following the instructions as in the screenshots below.
  3. Select "Start a New Project"

  4. Provide a name for your Pantheon site (this will go in the URL).

  5. Click Continue, and then click Deploy for WordPress.

  6. It will take a few minutes to install WordPress on Pantheon.

  7. After the install is complete, click the Visit your Pantheon Dashboard.

  8. Now, you'll go through the familiar WordPress setup. Select English, and then click Continue.

  9. Provide the WordPress Site Title, your preferred Username and Password, and an email address for password recovery. If you want your website to be accessible to search engines, check the box at the bottom. Then click Install WordPress.

  10. Afterward, you should be directed to the WordPress dashboard.

  11. Congrats! You set up your first WordPress site that is live on the web.

Creating a second WordPress site on Patheon for the AHR project.

  1. Navigate to the Pantheon Dashboard from your Pantheon Site Dashboard (distinct from the WordPress Dashboard), by clickingon your Username in the upper right corner, and clicking on My Dashboard.
  2. This will open up your Pantheon Dashboard, where you can see how many sites you have set up in Pantheon. You can have up to two "sandbox" sites for free. The first site, which we just created will eventually be used for your Portfolio. We're going to create your second site for the AHR project.
  3. In the Sites tab, click the Create New Site button.
  4. Pick a name for your AHR site. It should include AHR in it, but it needs to be unique across the internet (and our class), so you may want to pick something like your name, followed by AHR. Then click Continue.
  5. Return to step 5 of the previous section, and set up WordPress for the AHR site.

Importing your WordPress site from MAMP

WordPress comes with some Import/Export functionality that can be used to copy a WordPress instance from one location to another, such as MAMP to Pantheon.

The downside is: you will need to re-install all of the plugins you used on MAMP on your Pantheon WordPress sites.

AND you'll also need to re-upload all of your images to your Pantheon WordPress sites.

  1. Navigate to the MAMP WordPress Dashboard of the site you want to Export
  2. In the sidenav, under Tools, click Export. Under "Choose What to Export", make sure "All content" is selected. Then click Download Export File.
  3. Remember where the export file was saved at. It will be named something like "sitename.WordPress.2022-07-08.xml"
  4. Navigate to the Pantheon WordPress Dashboard of the site you want to Import to.
  5. Install ALL of the active Plugins used in the WordPress site you're importing from.
  6. In the sidenav, under Tools, click Import. Then, under the WordPress section, click Install Now to install the Importer Tool.
  7. Then click the Run Importer link.
  8. Click "Choose File" to select the file we exported in steps 2 and 3. Then click "Upload file and import".
  9. After the upload and import, you'll see a status screen, describing what was imported. You will probably see a lot of messages about images being unable to be imported. That is to be expected.
  10. Upload all the images you used on the site you imported from (at least, for all the pages you want working), and replace all the broken images with images from the Media Library
  11. Congrats! You successfully imported a WordPress site from MAMP!
