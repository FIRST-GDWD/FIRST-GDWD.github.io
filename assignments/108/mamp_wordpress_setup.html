Installing MAMP

  1. Download MAMP from here
  2. Install MAMP. You should see multiple screens like the screenshots below. Make sure to grant MAMP any privileges it needs to run.

    You do not need to install MAMP PRO or Apple Bonjour.

    Accept the license agreement.

    Install at the default folder location.

    Leave the default for the shortcut as well.

    Check the box if you want a desktop icon.

    Confirm the settings, and then click Install.

    Click Finish once the installation process has completed.

  3. Start up MAMP.
  4. Click the Start Servers button. If successful, the button will change to green and say Stop Servers.
  5. Click the Open WebStart Page button. This will open up the start up page for MAMP.

Creating a Database Table for Wordpress

  1. On the WebStart Page for MAMP, navigate down to the phpMyAdmin link, and click on it.
  2. This will open up phpMyAdmin, where you can manipulate and create databases.
  3. Click on the Databases tab.
  4. Type 'wordpress' as the name for the new database, and click Create.

Installing WordPress

  1. Download Wordpress from here.
  2. The zip file was likely saved to your Downloads folder, unless you have configured otherwise.
  3. Unzip the WordPress zip file you downloaded.
  4. Locate the htdocs folder inside of MAMP (If you followed the instructions above, it should probably be at C://MAMP/htdocs).
  5. Move the folder called 'wordpress' (not 'wordpress-6.0', it should be inside of that) into the MAMP htdocs folder.
  6. Navigate to wordpress on your localhost.
    • On Windows, this should be found at: localhost/wordpress
    • On Mac, this should be found at: localhost:80/wordpress or localhost:8000/wordpress
  7. You should then see the following pages as you configure the install:

    Select English as your language, and click Continue

    This guide will provide you with the information you need here. Just click Let's go!

    Specify 'root' as the Username and Password. Everything else should be left as the defaults. Click Submit.

    Click Run the installation.

    Provide a Site Title, a personal Username and Password, and Email. Then click Install WordPress.

    If successful, you should see the following. Click Log In.

    After logging in, you should see the following.

  8. Congrats! You installed WordPress!
