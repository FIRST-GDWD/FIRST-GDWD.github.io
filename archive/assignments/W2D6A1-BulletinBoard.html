OBJECTIVES:

    • Create at least TWO (2) rows of post-its
      • each with at least THREE (3) post-its in each row
    • Practice using the CSS Properties you've learned today

 

NEW CONCEPTS WE'LL BE USING:

    • CSS Class Selectors
    • CSS Properties
      • display (block, inline, inline-block)

 

HOW TO COMPLETE:

Recently, we learned our first major layout building tool: the display property!  With this CSS property, we can now control the flow of the HTML elements on our page: we can specify whether they should sit side by side other elements (inline or inline-block) or whether they should take up an entire line on their own (block).

We also learned how to use class attributes in HTML along with class selectors in CSS to share the same CSS properties across multiple HTML elements.

Our goal for this exercise is to utilize these new skills to fill out a bulletin board with some post-its; it's up to you to remember how to use the display property to layout the post-its according to the assignment specifications!

 

(HINT: This demo is a good refresher on the principles you need to use in this assignment)

 

If you're unsure about how to unzip the project template below, or how to get it into your repository, check out the following tutorial from the Coding Cookbook - Importing a Code Template.

 

    1. Download and unzip the bulletin board template; place the unzipped folder into your repository with the name "bulletin-board".  It should contain:
      • a css folder
        • setup.css (don't modify this file)
        • style.css (this is the file you'll be writing your CSS in)
      • an images folder
        • corkboard.jpg
        • frame.jpg
      • index.html
    2. Your objective is to create at least TWO (2) rows of post-its inside the <div> element with the id of "bulletin-board"
      • and each row should contain at least THREE (3) post-its 
    3. To create a post-it,
      1. add a <div> element with a "post-it" class inside the bulletin board element
      2. fill the <div> element with whatever content you want to include on your board. 
        • Consider things like:
          • reminders
          • motivation quotes
          • pictures
        • Also remember you can use different HTML elements like:
          • headings (<h1> - <h6>)
          • paragraphs (<p>)
          • lists (<ul>, <ol>, and <li>)
          • images (<img>)
      3. In the CSS, create a rule for your "post-it" class, and set the following properties (some of the values are up to you to pick/figure out)
        • background-color
        • display
        • width
        • height
        • margin: 24px; (we haven't formally learned margin yet; just trust that this will work for now)
        • padding: 24px; (we haven't formally learned padding yet; just trust that this will work for now)
        • font-family
    4. Repeat steps 3A and 3B for additional post-its
      1. Consider setting up different color classes that you can apply to each post-it, to add some variety
      2. Remember that you can add multiple classes to an HTML element.  If we wanted to add a "yellow" class in addition to the "post-it" class, we put both of them in the class attribute, separated by a space, as in the code snippet below:
        • <div class="post-it yellow"></div>
    5. Remember that you need to have two rows of post-its; the steps above do not include how to build such a layout. 
      • Think back to what you learned about Document Flow today, and apply it to your post-its!
      • Also be sure to test your code in the browser; the preview window is often too narrow to truly test your layout (unless you're working on a really wide monitor).

 

When completed, your preview window in Visual Studio Code should resemble something like the image at the top of the assignment, just with your own content.  As long as you have those two rows of post-its, feel free to submit your code (make sure you test you code in the browser beforehand though!).

 

CHALLENGE

However, if you want to challenge yourself, there are a couple new CSS properties we could take advantage of to make our bulletin board look even more realistic, as in the image below:

 

 

The image above has subtle shadows added to the post-its, and also added rounded circles with their own shadows that resemble push pins.  The new CSS properties used in the image above are:

    • box-shadow
      • This CSS Property accepts many values, used to offset the shadow, blur it, and even color it
    • border-radius: 50%;
      • We'll learn more about this property soon, but for now, just trust us that when this code is applied to a square element, it will completely round the corners and turn it into a circle.

These extra properties are optional, but if you have the time, try and figure out how you can use them to modify your bulletin board!  They're good properties to add to your CSS toolkit.
