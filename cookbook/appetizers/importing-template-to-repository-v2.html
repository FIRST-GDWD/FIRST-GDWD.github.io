Importing a Code Template

A tutorial showing how to get new code into your repository

A couple assignments in this course require you to use code templates provided for you.

Usually these templates are either to ensure everyone's working on the same code, or to give you a more inspiring starting point by using code you haven't learned yet.

However, it can be a little tricky to figure out how to get these code templates into your code repository if you haven't done it before (or maybe you did, but forgot!).

Either way, read on! It's not too bad once you do it a few times.

Importing Code

Moving things from one folder to another

  1. First, you'll need to download whatever code template you need for your Exercise or Project.

    We'll use the Bulletin Board project as our example in this tutorial (bulletin-board.zip).

    Most likely, your downloaded code template ended up in the Downloads folder. It'll be a .zip file, which is just a compact file format typically used for sending over multiple files.

    (Depending on your computer settings, there's a chance you might not see the .zip after the file name.

    If so, don't worry; it's still the right file. In that case, you'll just have to identify the .zip file by its icon or the label in the "Kind" column, which would say "ZIP archive")

  2. Next, you'll need to unzip the zip file. This process will differ depending on whether you're on Mac or PC.

    Mac

    If you're on Mac, you got lucky! Just double-click on the zip file; the file will get unzipped and a new unzipped folder will appear next to it. Done!

    Windows/PC

    If you're on Windows, the process is a bit more involved...

    Right-click on the zip file, and select "Extract All".

    A dialog will pop up. Just leave the defaults as they are, and click the "Extract" button.

    Then you will have successfully unzipped the zip file! ...but Windows does something a little weird: the actual unzipped folder usually ends up inside ANOTHER folder with an identical name as the unzipped folder.

    Make sure that for the next step, you are working with the inside folder, not the one that is next to the zip file. (And again, this is only for Windows users! Not Mac users!)

  3. After unzipping the template's zip file, you should see a new folder with the same name as the zip file.

    Inside this folder is a set of folders and files, all of which we need inside of your code repository.

    To achieve this, we need to get your Finder window (for Mac users) or your File Explorer window (for Windows/PC users) on top of Visual Studio Code (or next to).

    Once you've done this, just click and drag the unzipped folder on top of the Projects folder in your repository (or whatever other folder you want to place it in).

  4. After that, you should see the unzipped folder in your repository, and if you open it, you should see the usual folders and files you'd expect to see inside of your repository.

    Double-check which files you should be modifying for the assignment, preview index.html as usual, and then get to coding!

Now you know how to import a code template into your repository! Good luck in your assignment!
