OBJECTIVES:

    • Duplicate your code-skeleton project folder in your repository
    • Create a hello-world project
    • Write your first lines of code in HTML

 

NEW CONCEPTS WE'LL BE USING:

    • HTML Tags
    • HTML Elements
    • HTML Heading Elements
    • HTML Paragraph Elements

 

HOW TO COMPLETE:

It is tradition to have the very first program (or in this case, web page) you write in a new programming language output the words "Hello World"; we're going to follow in this proud tradition and do just that!

 

    1. Copy and paste your code-skeleton project folder inside of your repository folder, and rename the duplicate hello-world
    2. Make the following changes and additions to index.html:
      1. Change the content inside of the <title> tags from "Document" to "Hello World!"
      2. Create a heading element (h1) with the content of "Hello World!"
        • e.g. <h1>Hello World!</h1>
      3. Create a paragraph element underneath your heading element, with the content of "-FirstName LastName",
        • but replace FirstName and LastName with your own first and last name!
        • e.g. <p>- Alex Wellworth</p>

 

CHALLENGE

That's it for this exercise! But if you want to experiment with some more with the HTML elements we learned about today, feel free to add them in! Just be sure to have the "Hello World!" and your name in there properly when you submit it!
