HOW TO COMPLETE:

There's a LOT of setup for this class; we use a good number of different applications and technologies that work together so that we can code a web site and share it with others.  The goal of this assignment is to get all those different apps and technologies set up, and also create our "Code Skeleton" project template that we'll be reusing frequently for our exercises and projects in the course. 

We call it a "Code Skeleton" because it's the bare bones of a web page; it's just enough to set up a functional page, but you'll need to fill in the gaps with content and code to really flesh out the page and make it more lively.  Okay, okay, the puns will end here.  Maybe.
