This paragraph has its background-color set to light pink via the hexadecimal value #FFCCCC, but this text inside has its own background-color set using the color name "yellow" via the CSS Rule with the Class Selector targeting the span element. However, the background-color of the web page's body element is set to the color name "lavender" via an HTML Element Selector.

All paragraphs inside of a div have their background-color property set to the color "green".

Notice that the background-color only fills up to where the border of the element is (remember the different parts of the Box Model!).

To make elements look more aesthetically pleasing, we usually want to add padding to an element with a background-color, in order to give the content inside some breathing room, unless we're going for a "highlighter" type of effect like on this text.

Also notice that adding margin does not increase the area of the background-color around the text; only padding does that.
