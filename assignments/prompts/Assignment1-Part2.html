Cloning your repository via the Github Desktop app

  1. If you haven't already, download and install GitHub Desktop from the following site: https://desktop.github.com/

  2. After installation, log in using your new GitHub account.  It will take you briefly back to the browser to confirm your identify; click "Allow" when prompted.  You should see some pages in the app like the following:

    Here's the page you will see when it redirects you to Github to authenticate yourself.



    On the Configure Git, select the "Use my Github account name and email address".


  3. After you've successfully logged in to GitHub Desktop, we are going to "clone" your remote repository to your local machine.  What this means is, we are going to pull down and create a copy of the repository from Github.com (the remote location) to your Desktop (the local location).
  4. Create a new folder on your desktop called "GitHub".
  5. In GitHub Desktop, click the "Clone a Repository from the Internet" button.



  6. Select the Repository we just created earlier on Github.com, and select the folder on your Desktop you just created (you may need to add your_username.github.io to the file path afterwards if it doesn't auto-populate it; see the screenshot below).



  7. After clicking "Clone", Github Desktop will download your repository from the web.  After the download, you should see a new folder named after your repository, with a readme file inside it.  This means you successfully cloned your repository!
